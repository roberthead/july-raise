# CLAUDE.md

## What this repo is

A working space to organize resources and thoughts for the July 17–19 "Rogue Raise"
event at White Rabbit in Ashland, Oregon. The substance so far lives in `docs/` —
meeting transcripts, contacts, presentation notes, and homelessness PIT-count
material.

A fresh Rails 8 application is scaffolded underneath (Ruby 3.4.4, Rails ~> 8.1,
Propshaft + importmap + Turbo/Stimulus, Minitest, Kamal/Docker for deploy), but no
domain features have been built yet. Don't assume an "app" exists to describe —
treat `app/`, `db/`, etc. as an untouched generator skeleton until code shows up.

## Layout

- `docs/` — the actual content: notes, transcripts, contacts, presentation drafts.
- Rails directories (`app/`, `config/`, `db/`, `test/`, …) — default scaffold.

## Conventions

- Markdown: blank lines before and after every header.
